# 메모
## 실행(macOS)
- xQuarts 설치해야됨 (GUI 프로그램이기 때문)
- 설치 이후 터미널에 아래 명령어 입력
```
xhost + 127.0.0.1
```
- 아래 명령어로 실행
```
docker-compose up --build
```
